
# Blueprint: AI 동물상 테스트 & 메뉴 추천 사이트

## 개요

이 애플리케이션은 사용자가 자신의 사진을 업로드하여 강아지상/고양이상 여부를 테스트하고, 저녁 메뉴를 추천받으며, 제휴 문의를 보낼 수 있는 통합 서비스 웹사이트입니다.

## 기능

-   **AI 동물상 테스트:** Teachable Machine 모델을 사용하여 업로드된 이미지에서 강아지상과 고양이상의 확률을 분석하고 결과를 표시합니다.
-   **메뉴 추첨:** 사용자는 버튼을 클릭하여 저녁 메뉴를 무작위로 추첨할 수 있습니다.
-   **테마 전환:** 다크 모드와 라이트 모드 간의 전환이 가능하며, 사용자의 설정이 브라우저에 저장됩니다.
-   **제휴 문의:** Formspree를 연동하여 사용자가 제휴 문의를 보낼 수 있는 폼을 제공합니다.

## 디자인 및 스타일

-   **레이아웃:** 섹션별로 구분된 깔끔하고 현대적인 중앙 집중형 디자인입니다.
-   **시각화:** AI 분석 결과를 게이지 바(Gauge Bar) 형식으로 시각화하여 전달력을 높였습니다.
-   **테마:** CSS 변수를 활용하여 시스템 및 사용자 선호 테마에 대응합니다.
-   **반응형:** 다양한 기기에서 최적화된 화면을 제공합니다.

## 기술 스택

-   **HTML/CSS/JS:** 프레임워크 없이 바닐라 기술로 구축되었습니다.
-   **TensorFlow.js & Teachable Machine:** 브라우저 내 AI 모델 실행을 담당합니다.
-   **Formspree:** 이메일 기반 문의 처리 시스템입니다.
